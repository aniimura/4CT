第11章

保存則

このページで学ぶこと

  • ケンペ交換で何が保存されるか
  • 補題1（保存則）の正確な主張
  • 辺数と頂点数の保存
  • サイクルランクと成分数の関係式
前提: 第3章（ケンペ交換）、第5章（サイクルランク）

なぜ保存則が必要か

B4彩色から非連結 $\{d,\alpha\}$-交換を行うと、新しい彩色 $\varphi'$ が得られます。 この新しい彩色の構造（成分数やサイクルランク）を知りたいのですが、 直接計算するのは難しい。そこで「交換の前後で変わらないもの（保存量）」を見つけ、 間接的に新しい彩色の構造を推論します。

物理学のアナロジー

物理学ではエネルギー保存則が重要です：反応前後でエネルギーの総量は変わらない。 同様に、ケンペ交換の前後で「辺と頂点の特定の組み合わせ」が保存されます。 この保存量が、交換後の状態を制約する強力なツールとなります。

補題1：保存則

補題1（保存則）. $\{d,\alpha\}$-ケンペ交換のもとで、$d, \alpha$ 以外の任意の色 $\xi$ に対し： $$e'_{d,\xi} + e'_{\alpha,\xi} = e_{d,\xi} + e_{\alpha,\xi}$$ $$n'_d + n'_\alpha = n_d + n_\alpha$$ ここで $e_{a,b}$ は $H_{a,b}$ の辺数、$n_c$ は色 $c$ の頂点数。

何が言われているか

$\{d,\alpha\}$-交換は、鎖 $C$ 上で $d$ と $\alpha$ を入れ替えます。 このとき、第三の色 $\xi$（例えば $\gamma$ や $\beta$）から見ると：

  • 辺の保存：「$d$-$\xi$ の辺」と「$\alpha$-$\xi$ の辺」の合計は変わらない
  • 頂点の保存：「$d$ 色の頂点」と「$\alpha$ 色の頂点」の合計は変わらない

個々の $e_{d,\xi}$ や $e_{\alpha,\xi}$ は変化しますが、その和は不変です。

証明

証明.
頂点の保存. 交換する鎖 $C$ 上で、$d$ 色の頂点が $\alpha$ 色に変わり、 $\alpha$ 色の頂点が $d$ 色に変わる。 $C$ 外の頂点の色は変わらない。 よって $n'_d + n'_\alpha = n_d + n_\alpha$（$d$ と $\alpha$ の総数は同じ）。
辺の保存. 色 $\xi$ の頂点 $u$ を考える（$u$ は $C$ に含まれない ── $\xi \neq d, \alpha$ だから）。 $u$ の近傍のうち $C$ に属する頂点 $w$ は、色が $d \to \alpha$ または $\alpha \to d$ に変わる。
  • 元々 $w$ が $d$ 色で、辺 $\{u,w\}$ が $H_{d,\xi}$ に属していたなら、 交換後は $w$ が $\alpha$ 色になり、辺 $\{u,w\}$ は $H_{\alpha,\xi}$ に属する
  • 元々 $w$ が $\alpha$ 色なら、辺は $H_{\alpha,\xi}$ から $H_{d,\xi}$ に移動する
つまり各辺は $H_{d,\xi}$ と $H_{\alpha,\xi}$ の間を「移動」するだけで、 新しく生まれたり消えたりはしない。よって $e'_{d,\xi} + e'_{\alpha,\xi} = e_{d,\xi} + e_{\alpha,\xi}$。
$\QED$

サイクルランクへの帰結

保存則をサイクルランクの定義 $L = E - V + p$ に代入すると：

系（サイクルランクの保存）. $$L'_{d,\xi} + L'_{\alpha,\xi} = L_{d,\xi} + L_{\alpha,\xi} + (p'_{d,\xi} + p'_{\alpha,\xi}) - (p_{d,\xi} + p_{\alpha,\xi})$$
導出. $L_{d,\xi} = e_{d,\xi} - (n_d + n_\xi) + p_{d,\xi}$ であるから： $$L_{d,\xi} + L_{\alpha,\xi} = (e_{d,\xi} + e_{\alpha,\xi}) - (n_d + n_\alpha) - 2n_\xi + (p_{d,\xi} + p_{\alpha,\xi})$$ 交換後も同様の式が成り立ち、辺数と頂点数の和は保存されるので、差をとると上の式が得られる。 $\QED$

この式の使い方

$L \geq 0$（サイクルランクは非負）であることを使って、成分数の和に下界を与えます：

$$L'_{d,\xi} + L'_{\alpha,\xi} \geq 0$$ $$\implies p'_{d,\xi} + p'_{\alpha,\xi} \geq p_{d,\xi} + p_{\alpha,\xi} - L_{d,\xi} - L_{\alpha,\xi}$$

B4彩色では $L_{d,\beta} = L_{\alpha,\beta} = 0$（定理6）なので、$\xi = \beta$ の場合：

$$p'_{d,\beta} + p'_{\alpha,\beta} \geq p_{d,\beta} + p_{\alpha,\beta} = 2 + p_{\alpha,\beta} \geq 3$$

この不等式が次章の B4非再帰定理の鍵となります。

具体例

B4彩色からの非連結 $\{d,\alpha\}$-交換

B4の設定（定理6）で、$\xi = \beta$ として保存則を適用：

交換前交換後保存量
$e_{d,\beta} + e_{\alpha,\beta}$$e + e'$$e'' + e'''$ $e + e' = e'' + e'''$ ✓
$n_d + n_\alpha$$n + m$$n' + m'$ $n + m = n' + m'$ ✓
$p_{d,\beta} + p_{\alpha,\beta}$$2 + p_{\alpha,\beta}$ $p'_{d,\beta} + p'_{\alpha,\beta}$変化しうる
$L_{d,\beta} + L_{\alpha,\beta}$$0 + 0 = 0$ $L'_{d,\beta} + L'_{\alpha,\beta} \geq 0$増加しうる

辺と頂点の和は保存される。サイクルランクは増加しうるが非負。 成分数の和に下界：$p'_{d,\beta} + p'_{\alpha,\beta} \geq 2 + p_{\alpha,\beta} \geq 3$。

ポイント

  • 補題1：$\{d,\alpha\}$-交換で、第三色 $\xi$ に対する辺数の和と頂点数の和は保存される。
  • サイクルランクの和の変化は成分数の和の変化に等しい。
  • $L \geq 0$ を使うことで、成分数の和に下界が得られる。
  • B4彩色では $L_{d,\beta} = L_{\alpha,\beta} = 0$（定理6）なので、$p'_{d,\beta} + p'_{\alpha,\beta} \geq 3$。
  • この下界が B4非再帰定理（次章）の証明の核心。
