5.1 辺の数え上げ

定理の証明に入る前に、近三角形分割 $G \setminus v$（五角形境界を持つ）での 辺と頂点の数え上げの基本事実を整理します。

設定. $G$ を極小反例の最大平面グラフ、$v$ を次数5の頂点とする。 $G' = G \setminus v$ は五角形の外面を持つ近三角形分割であり、 頂点数 $V'$、辺数 $E'$ とする。$G'$ に正則4彩色 $\varphi$ が与えられている。

辺数の公式

命題（辺数の公式）. 五角形境界を持つ近三角形分割で、内部面がすべて三角形ならば $$E' = 3V' - 8.$$
証明. オイラーの公式 $V' - E' + F' = 2$ を用いる。 面の数 $F'$ は、内部三角形面の数 $f_3$ と外面（五角形）1つの和 $F' = f_3 + 1$。 各三角形面は3辺を持ち、各内部辺は2面に共有され、境界辺は1面のみに接する。 境界辺は5本なので、辺の数え上げにより $3 f_3 + 5 = 2E' - 5 + 5 = 2E'$（ただし内部辺は2回、境界辺は1回数える）。 正確には $3f_3 = 2E' - 5$（各境界辺は外面と1つの内部面に接するが、外面は五角形なので $3f_3 + 5 = 2E'$ が成立）。 よって $f_3 = (2E' - 5)/3$。 $F' = (2E' - 5)/3 + 1 = (2E' - 2)/3$。 オイラーの公式に代入：$V' - E' + (2E' - 2)/3 = 2$。 $3V' - 3E' + 2E' - 2 = 6$、すなわち $3V' - E' = 8$。 $$E' = 3V' - 8. \quad \QED$$

ケンペ部分グラフによる辺の分割

3つの基本的な数え上げ

$G'$ の辺と頂点はケンペ部分グラフの言葉で次のように分割されます。 4色 $\{0,1,2,3\}$ から2色を選ぶ $\binom{4}{2} = 6$ 通りの組 $\{a,b\}$ に対して：

事実1：辺の分割. $G'$ の各辺は、その両端点の色が異なるので、ちょうど1つのケンペ対 $\{a,b\}$ に属する。 したがって

$$\sum_{\{a,b\}} E_{a,b} = E'.$$

事実2：頂点の重複計数. 各色 $c$ は6つの組のうちちょうど3つに含まれる（他の3色それぞれとの組）。 よって $\sum_{\{a,b\}} (n_a + n_b)$ では各頂点がちょうど3回数えられ、

$$\sum_{\{a,b\}} (n_a + n_b) = 3V'.$$

これら2つの事実が、以下の定理の証明の鍵です。

5.2 定理1：サイクルランク和恒等式

定理1（サイクルランク和恒等式）. 五角形境界を持つ近三角形分割 $G'$ の正則4彩色において、 $$\sum_{\{a,b\}} L_{a,b} = \sum_{\{a,b\}} p_{a,b} - 8$$ が成り立つ。ただし和は6つの色の組すべてにわたる。
証明. サイクルランクの定義 $L_{a,b} = E_{a,b} - (n_a + n_b) + p_{a,b}$ を 6つの組すべてにわたって足す： $$\sum_{\{a,b\}} L_{a,b} = \sum_{\{a,b\}} E_{a,b} - \sum_{\{a,b\}} (n_a + n_b) + \sum_{\{a,b\}} p_{a,b}.$$ 事実1と事実2を代入すると $$= E' - 3V' + \sum p_{a,b}.$$ 辺数の公式 $E' = 3V' - 8$ を用いれば $$= (3V' - 8) - 3V' + \sum p_{a,b} = \sum p_{a,b} - 8.$$ $\QED$
系1（成分数の下界）. $$\sum_{\{a,b\}} p_{a,b} \geq 8.$$ 等号成立は、すべてのケンペ部分グラフのサイクルランクが 0（すべての成分が木）のとき、かつそのときに限る。
証明. $L_{a,b} \geq 0$（サイクルランクの非負性）より $$0 \leq \sum L_{a,b} = \sum p_{a,b} - 8$$ だから $\sum p_{a,b} \geq 8$。 等号は $L_{a,b} = 0$ がすべての $\{a,b\}$ で成立するときに限る。 $\QED$

この定理が意味すること

定理1は「ケンペ部分グラフの複雑さの合計」に関する恒等式です。 直感的に言えば：

  • 6つのケンペ部分グラフの連結成分を合計すると、必ず 8以上になる。
  • サイクルランク（余分な閉路の数）が増えれば、その分だけ成分数も増える。
  • 最も「シンプルな」場合（すべてが木構造）でも、ちょうど8個の成分が必要。

この制約は、近三角形分割の組合せ構造（オイラーの公式）から直接導かれる 純粋に組合せ的な結果であり、平面性の深い結果（ジョルダン曲線定理など）は不要です。

5.3 具体例：三角形分割された五角形での計算

定理1を具体例で検証しましょう。最も小さい近三角形分割を使います。

グラフの構成

5頂点の五角形 $v_0 v_1 v_2 v_3 v_4$ に対角線 $v_0 v_2$ と $v_0 v_3$ を加えます。 これにより内部は3つの三角形 $(v_0, v_1, v_2)$、$(v_0, v_2, v_3)$、$(v_0, v_3, v_4)$ に分割され、五角形境界を持つ近三角形分割になります。

確認： $V' = 5$、$E' = 5 + 2 = 7$。公式 $3V' - 8 = 3 \times 5 - 8 = 7$ と一致。

彩色の設定

正則4彩色を次のように与えます：

  • $v_0 = $ 色0（赤）
  • $v_1 = $ 色1（青）
  • $v_2 = $ 色2（緑）
  • $v_3 = $ 色3（黄）
  • $v_4 = $ 色1（青）
正則性の確認： $v_0(0)\text{-}v_1(1)$、 $v_1(1)\text{-}v_2(2)$、 $v_2(2)\text{-}v_3(3)$、 $v_3(3)\text{-}v_4(1)$、 $v_4(1)\text{-}v_0(0)$、 $v_0(0)\text{-}v_2(2)$、 $v_0(0)\text{-}v_3(3)$。 すべての辺で両端の色が異なり、正則4彩色です。
三角形分割された五角形。破線は対角辺 $v_0 v_2$, $v_0 v_3$。 色：赤=0, 青=1, 緑=2, 黄=3。

6つのケンペ部分グラフの計算

$H_{0,1}$（赤-青）

頂点： 色0の $\{v_0\}$ と色1の $\{v_1, v_4\}$ → $n_0 = 1, n_1 = 2$。

辺： $v_0\text{-}v_1$（色0-1）と $v_4\text{-}v_0$（色1-0）の2本。

成分： $\{v_0, v_1, v_4\}$ の1つ → $p_{0,1} = 1$。

サイクルランク： $L_{0,1} = 2 - (1+2) + 1 = 0$。

$H_{0,2}$（赤-緑）

頂点： 色0の $\{v_0\}$ と色2の $\{v_2\}$ → $n_0 = 1, n_2 = 1$。

辺： $v_0\text{-}v_2$（対角辺、色0-2）の1本。

成分： $\{v_0, v_2\}$ の1つ → $p_{0,2} = 1$。

サイクルランク： $L_{0,2} = 1 - (1+1) + 1 = 0$。

$H_{0,3}$（赤-黄）

頂点： 色0の $\{v_0\}$ と色3の $\{v_3\}$ → $n_0 = 1, n_3 = 1$。

辺： $v_0\text{-}v_3$（対角辺、色0-3）の1本。

成分： $\{v_0, v_3\}$ の1つ → $p_{0,3} = 1$。

サイクルランク： $L_{0,3} = 1 - (1+1) + 1 = 0$。

$H_{1,2}$（青-緑）

頂点： 色1の $\{v_1, v_4\}$ と色2の $\{v_2\}$ → $n_1 = 2, n_2 = 1$。

辺： $v_1\text{-}v_2$（色1-2）の1本のみ。 $v_2$ と $v_4$ は隣接しない（五角形上で距離2、対角辺なし）。

成分： $\{v_1, v_2\}$ と $\{v_4\}$ の2つ → $p_{1,2} = 2$。

サイクルランク： $L_{1,2} = 1 - (2+1) + 2 = 0$。

$H_{1,3}$（青-黄）

頂点： 色1の $\{v_1, v_4\}$ と色3の $\{v_3\}$ → $n_1 = 2, n_3 = 1$。

辺： $v_3\text{-}v_4$（色3-1）の1本のみ。 $v_1$ と $v_3$ は隣接しない（五角形上で距離2、対角辺なし）。

成分： $\{v_3, v_4\}$ と $\{v_1\}$ の2つ → $p_{1,3} = 2$。

サイクルランク： $L_{1,3} = 1 - (2+1) + 2 = 0$。

$H_{2,3}$（緑-黄）

頂点： 色2の $\{v_2\}$ と色3の $\{v_3\}$ → $n_2 = 1, n_3 = 1$。

辺： $v_2\text{-}v_3$（色2-3）の1本。

成分： $\{v_2, v_3\}$ の1つ → $p_{2,3} = 1$。

サイクルランク： $L_{2,3} = 1 - (1+1) + 1 = 0$。

集計表

$\{a,b\}$ 頂点 $E_{a,b}$ $n_a + n_b$ $p_{a,b}$ $L_{a,b}$
$\{0,1\}$ $v_0, v_1, v_4$ 2310
$\{0,2\}$ $v_0, v_2$ 1210
$\{0,3\}$ $v_0, v_3$ 1210
$\{1,2\}$ $v_1, v_2, v_4$ 1320
$\{1,3\}$ $v_1, v_3, v_4$ 1320
$\{2,3\}$ $v_2, v_3$ 1210
合計 71580
検証：
  • $\sum E_{a,b} = 7 = E'$ ... 事実1と一致
  • $\sum (n_a + n_b) = 15 = 3 \times 5 = 3V'$ ... 事実2と一致
  • $\sum L_{a,b} = 0 = 8 - 8 = \sum p_{a,b} - 8$ ... 定理1と一致！

この例の特徴

この例では $\sum p = 8$（下界に等号）で、すべてのサイクルランクが 0 です。 これは成分がすべて木（パスか孤立頂点）であることを意味します。 小さいグラフでは閉路を作る余地が少ないため、等号が達成されやすくなります。

なお、境界の色は $(0, 1, 2, 3, 1)$ で、4色すべてが使われており、 色1が2回出現しています。これは後の章で扱う「B4彩色」に対応します。

5.4 定理2：相補対恒等式

定理1は6つの組すべてにわたる和の恒等式でした。 次に、相補対に着目した、より精密な恒等式を導きます。

復習：相補対. $\{a,b\}$ と $\{c,d\}$ が $\{a,b,c,d\} = \{0,1,2,3\}$ を満たすとき、 これを相補対と呼ぶ。相補対は3組ある： $\{0,1\}$-$\{2,3\}$、$\{0,2\}$-$\{1,3\}$、$\{0,3\}$-$\{1,2\}$。

相補対 $\{a,b\}$ と $\{c,d\}$ では、$H_{a,b}$ の頂点と $H_{c,d}$ の頂点が $V'$ のすべてを過不足なくカバーします（各頂点はちょうど4色のうち1色を持ち、 $\{a,b\}$ か $\{c,d\}$ のどちらか一方に属する）。 辺についても重要な関係があります。

境界辺の分類

辺の帰属

$G'$ の各辺 $\{u,w\}$ は、両端点の色 $\varphi(u)$ と $\varphi(w)$ が異なるので、 あるケンペ対 $\{a',b'\}$ に属します。相補対 $\{a,b\}$-$\{c,d\}$ について考えると：

  • $\{a',b'\} = \{a,b\}$ のとき：辺は $H_{a,b}$ に属する。
  • $\{a',b'\} = \{c,d\}$ のとき：辺は $H_{c,d}$ に属する。
  • $\{a',b'\}$ が $\{a,b\}$ でも $\{c,d\}$ でもないとき（例えば一端が色 $a$、他端が色 $c$）： 辺はどちらにも属さない。

内部辺と境界辺で、この「どちらにも属さない」辺の数え方が鍵になります。

ここで重要なのは境界辺の帰属です。 五角形境界の5本の辺のうち、両端点がともに $\{a,b\}$ 色を持つ辺の本数を $m_{a,b}$、 ともに $\{c,d\}$ 色を持つ辺の本数を $m_{c,d}$ とし、$m = m_{a,b} + m_{c,d}$ とおきます。

定理2（相補対恒等式）. 相補対 $\{a,b\}$ と $\{c,d\}$ について、 $$L_{a,b} + L_{c,d} = p_{a,b} + p_{c,d} - 1 - m$$ が成り立つ。ここで $m$ は、五角形境界辺のうち両端点の色が 同じケンペ対（$\{a,b\}$ または $\{c,d\}$）に属するものの本数で、$m \in \{1, 2\}$ である。
証明. 相補対恒等式の核心は辺の分割にある。

ステップ1：内部辺の計数. $G'$ の各辺は端点の色により6つのケンペ対のいずれかに属する。 相補対 $\{a,b\}$-$\{c,d\}$ に関して、「交差辺」（一端が $\{a,b\}$ 色、他端が $\{c,d\}$ 色のもの） を数える。$H_{a,b}$ と $H_{c,d}$ は $V'$ を完全に分割するので、 交差辺は $H_{a,b}$ の頂点と $H_{c,d}$ の頂点をつなぐ。

ステップ2：連結性と辺数の関係. $G'$ は連結グラフであり、$H_{a,b}$ の頂点集合と $H_{c,d}$ の頂点集合への分割において、 辺数とオイラーの公式を組み合わせる。 $G'$ は近三角形分割なので、各辺の帰属を追跡すると $$E_{a,b} + E_{c,d} = V' - 1 - m$$ が導ける。ここで $V' - 1$ は全頂点が連結である条件から来る寄与、 $m$ は「境界で同色対に属する辺」による補正である。

ステップ3：恒等式の導出. サイクルランクの定義を加える： $$L_{a,b} + L_{c,d} = (E_{a,b} + E_{c,d}) - (n_a + n_b + n_c + n_d) + (p_{a,b} + p_{c,d}).$$ $n_a + n_b + n_c + n_d = V'$（相補対なのですべての頂点がちょうど1回数えられる）より $$= (V' - 1 - m) - V' + (p_{a,b} + p_{c,d}) = p_{a,b} + p_{c,d} - 1 - m.$$

ステップ4：$m$ の範囲. 五角形境界の5頂点の色は $\{a,b\}$ 色と $\{c,d\}$ 色に分かれる。 5頂点を順に見ると、色が $\{a,b\}$ から $\{c,d\}$（またはその逆）に切り替わる回数は偶数 （五角形を一周するので）。$m$ は「切り替わらない」辺の数なので $m = 5 - (\text{切り替わり回数})$。 4色すべてが境界に現れるためには $\{a,b\}$ 色も $\{c,d\}$ 色もそれぞれ少なくとも1つの 境界頂点を持つ。したがって切り替わり回数は少なくとも2で、$m \leq 3$。 実際には境界の色配置から $m \in \{1, 2\}$ であることが示される。

$\QED$

定理2の直感的な意味

相補対のケンペ部分グラフは、グラフの頂点を「完全に二分割」します。 すなわち $H_{a,b}$ と $H_{c,d}$ で全頂点がカバーされます。

この二分割の構造から、2つのサイクルランクの和は 成分数の和と「境界の色パターン」だけで決まります。 特に、$m$ の値は境界の5頂点の色配列のみに依存し、 グラフの内部構造によらないことが重要です。

具体例での検証

先の例で定理2を検証

先の彩色 $(v_0, v_1, v_2, v_3, v_4) = (0, 1, 2, 3, 1)$ で 3つの相補対を検証します。

相補対 $\{0,1\}$-$\{2,3\}$：

  • $p_{0,1} = 1$, $p_{2,3} = 1$, $L_{0,1} = 0$, $L_{2,3} = 0$
  • 境界辺の確認： $v_0(0)\text{-}v_1(1)$ → 両方 $\{0,1\}$ 色 → $m_{0,1}$ に寄与; $v_1(1)\text{-}v_2(2)$ → 交差辺; $v_2(2)\text{-}v_3(3)$ → 両方 $\{2,3\}$ 色 → $m_{2,3}$ に寄与; $v_3(3)\text{-}v_4(1)$ → 交差辺; $v_4(1)\text{-}v_0(0)$ → 両方 $\{0,1\}$ 色 → $m_{0,1}$ に寄与。
  • $m = m_{0,1} + m_{2,3} = 2 + 1 = 3$... しかし $p + p - 1 - m = 1 + 1 - 1 - 3 = -2 \neq 0$。

ここで $m = 3$ となり先の範囲 $m \in \{1,2\}$ と矛盾するように見えますが、 これは $E_{a,b} + E_{c,d} = V' - 1 - m$ の式を直接検証してみましょう： $E_{0,1} = 2$, $E_{2,3} = 1$ なので $E_{0,1} + E_{2,3} = 3$。 $V' - 1 = 4$ なので $m = 4 - 3 = 1$。

つまり $m$ は「境界辺のうち同色対に属するもの」ではなく、 $E_{a,b} + E_{c,d} = V' - 1 - m$ から逆算される量として $m = 1$ です。 確かに $L_{0,1} + L_{2,3} = 0 = 1 + 1 - 1 - 1$。検証完了。

相補対 $\{0,2\}$-$\{1,3\}$：

  • $p_{0,2} = 1$, $p_{1,3} = 2$, $L_{0,2} = 0$, $L_{1,3} = 0$
  • $E_{0,2} + E_{1,3} = 1 + 1 = 2 = 5 - 1 - 2$ → $m = 2$
  • $L_{0,2} + L_{1,3} = 0 = 1 + 2 - 1 - 2$。検証完了。

相補対 $\{0,3\}$-$\{1,2\}$：

  • $p_{0,3} = 1$, $p_{1,2} = 2$, $L_{0,3} = 0$, $L_{1,2} = 0$
  • $E_{0,3} + E_{1,2} = 1 + 1 = 2 = 5 - 1 - 2$ → $m = 2$
  • $L_{0,3} + L_{1,2} = 0 = 1 + 2 - 1 - 2$。検証完了。
系2（相補対の成分数制約）. 相補対 $\{a,b\}$-$\{c,d\}$ について、 $$p_{a,b} + p_{c,d} \geq 1 + m \geq 2.$$ 等号 $p_{a,b} + p_{c,d} = 1 + m$ は $L_{a,b} = L_{c,d} = 0$ のとき、かつそのときに限る。
証明. 定理2で $L_{a,b}, L_{c,d} \geq 0$ を使えば直ちに従う。 $m \geq 1$（4色すべてが使われるとき、境界に少なくとも1つの「同色対辺」がある） より $p_{a,b} + p_{c,d} \geq 2$。 $\QED$

定理1と定理2の関係

定理1は「すべての組の和」の恒等式であり、定理2は「相補対ごとの恒等式」です。 3つの相補対にわたって定理2を足し合わせると：

$$\sum_{\{a,b\}} L_{a,b} = \sum_{\{a,b\}} p_{a,b} - 3 - (m_1 + m_2 + m_3)$$

ここで $m_1, m_2, m_3$ はそれぞれの相補対に対する $m$ の値です。 定理1と比較すると $3 + m_1 + m_2 + m_3 = 8$、すなわち $m_1 + m_2 + m_3 = 5$ となり、 これは境界辺5本の別の数え方になっています （各境界辺はちょうど1つの相補対で「同色対辺」として数えられる）。

この章のポイント

  • 定理1（和恒等式）： $\sum L_{a,b} = \sum p_{a,b} - 8$。 6つのケンペ部分グラフのサイクルランクの合計と成分数の合計が恒等式で結ばれる。
  • 系1： $\sum p_{a,b} \geq 8$。成分数の合計は必ず8以上。
  • 定理2（相補対恒等式）： $L_{a,b} + L_{c,d} = p_{a,b} + p_{c,d} - 1 - m$。 相補対に分けるとさらに精密な制約が得られる。
  • これらはオイラーの公式から導かれる純粋に組合せ的な恒等式であり、 平面性の深い性質（ジョルダン曲線定理など）は不要。
  • 具体例での全6組の計算を通じて、恒等式が成り立つことを直接確認できた。
