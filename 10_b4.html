B4彩色の復習

第4章で導入した概念を復習します。

定義（B4彩色）. 近三角形分割 $G \setminus v$ の正則4彩色 $\varphi$ が B4彩色 であるとは：
  1. $\conn = 1$（非 $d$ 色のうちちょうど1つが連結色）
  2. 唯一の連結色 $\gamma$ が全域的（$p_{d,\gamma} = 1$）

標準的な設定

B4彩色では、一般性を失わず以下のように固定できます：

  • $b_1 = v_0$, $b_2 = v_2$：境界対（ともに色 $d$）
  • $v_1$：色 $\gamma$（唯一の連結色、$v_0$ と $v_2$ を結ぶ）
  • $v_3$：色 $\alpha$（非連結色の一つ）
  • $v_4$：色 $\beta$（非連結色のもう一つ）
B4彩色の構造。$v_0$ と $v_2$ は色 $d$ で、$\gamma$ 経由で $H_{d,\gamma}$ 内で連結。

定理6：B4構造定理

定理6（B4構造定理）. B4彩色において、境界 $v_0 = d, v_1 = \gamma, v_2 = d, v_3 = \alpha, v_4 = \beta$ のもとで： $$p_{d,\gamma} = 1, \quad p_{d,\alpha} = 2, \quad p_{d,\beta} = 2$$ $$p_{\alpha,\gamma} = 1, \quad p_{\gamma,\beta} = 1$$ $$L_{d,\alpha} = L_{d,\beta} = L_{\alpha,\beta} = 0$$

定理の意味

B4彩色では、6つのケンペ部分グラフの成分数がほぼ完全に決まります。 これは「最も困難なケースの構造は実は高度に規則的」ということを示しています。

ペア$p$$L$解釈
$\{d,\gamma\}$1—連結（B4の定義そのもの）
$\{d,\alpha\}$20ちょうど2成分、すべて木
$\{d,\beta\}$20ちょうど2成分、すべて木
$\{\gamma,\beta\}$10連結、相補のため森
$\{\alpha,\gamma\}$10連結、相補のため森
$\{\alpha,\beta\}$—0森

証明

証明.
ステップ1：$p_{d,\gamma} = 1$. B4の定義より、$\gamma$ は全域的連結色なので $p_{d,\gamma} = 1$。
ステップ2：$p_{d,\alpha} = 2$, $p_{\gamma,\beta} = 1$. $\alpha$ は非連結色なので、$b_1$ と $b_2$ は $H_{d,\alpha}$ の異なる成分にある。 よって $p_{d,\alpha} \geq 2$。

$\{d,\alpha\}$ と $\{\gamma,\beta\}$ は相補対。積の上界より：

$$p_{d,\alpha} \cdot p_{\gamma,\beta} \leq 2$$

$p_{d,\alpha} \geq 2$ なので $p_{\gamma,\beta} \leq 1$。$p \geq 1$ より $p_{\gamma,\beta} = 1$。 すると $p_{d,\alpha} \leq 2$。合わせて $p_{d,\alpha} = 2$。

ステップ3：$p_{d,\beta} = 2$, $p_{\alpha,\gamma} = 1$. $\beta$ も非連結色なので $p_{d,\beta} \geq 2$。 $\{d,\beta\}$ と $\{\alpha,\gamma\}$ は相補対。積の上界より： $$p_{d,\beta} \cdot p_{\alpha,\gamma} \leq 2$$ 同様の議論で $p_{\alpha,\gamma} = 1$, $p_{d,\beta} = 2$。
ステップ4：サイクルランク. 定理5 (iii) を適用：
  • $p_{\gamma,\beta} = 1 \implies L_{d,\alpha} = 0$
  • $p_{\alpha,\gamma} = 1 \implies L_{d,\beta} = 0$
  • $p_{d,\gamma} = 1 \implies L_{\alpha,\beta} = 0$
$\QED$

なぜB4が最も困難なケースか

ケンペの失敗

1879年のケンペは B4 ケースで致命的な誤りを犯しました。 ケンペは $\{d,\alpha\}$ と $\{d,\beta\}$ の2つの交換を同時に行おうとしました。 しかし1つ目の交換が2つ目のケンペ鎖の構造を壊してしまう可能性があり、 ヒーウッドが1890年にこの誤りを指摘しました。

本証明では一度に1回だけ交換を行い、 交換後の状態を代数的に追跡します。

B4の困難さは具体的に以下の点にあります：

  • $\gamma$ は全域的（$p_{d,\gamma} = 1$）なので $\{d,\gamma\}$-交換は境界全体に影響する
  • $\alpha$, $\beta$ は非連結だが、$\{d,\alpha\}$-交換が $\gamma$ と $\beta$ の構造を変えうる
  • 交換後に再び B4 になると、無限ループの危険がある

次章以降で、保存則と B4非再帰定理がこの問題をどう解決するかを見ます。

$p_{d,\alpha} = 2$ の意味

$H_{d,\alpha}$ がちょうど2つの成分に分かれるということは、 $b_1$ を含む成分と $b_2$ を含む成分が別々に存在するということです。 ケースCで行う「非連結 $\{d,\alpha\}$-交換」とは、 $b_1$ を含む成分だけを交換する（$b_2$ の成分には触れない）操作です。

$H_{d,\alpha}$ の2成分。非連結交換では成分1のみを交換する。

ポイント

  • 定理6：B4彩色の構造は積の上界によって完全に決定される。
  • 非連結色 $\alpha$, $\beta$ はそれぞれ $p = 2$（ちょうど2成分）。
  • 相補の成分は連結（$p = 1$）で森（$L = 0$）。
  • B4はケンペが失敗したケース。本証明は1回ずつ交換して代数的に追跡する。
  • $p_{d,\alpha} = 2$ であることが「非連結交換」を可能にする。
